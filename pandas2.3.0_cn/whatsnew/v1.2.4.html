1.2.4 版本新特性 (2021年4月12日)#

以下是 pandas 1.2.4 中的更改。有关包括其他 pandas 版本在内的完整更改日志，请参阅发布说明。

回归问题修复#

  • 修复了 DataFrame.sum() 中的回归问题，即当传入的 min_count 大于 DataFrame 的形状时会导致 ValueError (GH 39738)

  • 修复了 DataFrame.to_json() 中的回归问题，即在 PyPy 上运行时会引发 AttributeError (GH 39837)

  • 修复了 pd.NaT 与非日期时间型 NumPy 数组进行（不）相等比较时，返回标量而非数组的回归问题 (GH 40722)

  • 修复了 DataFrame.where() 中的回归问题，即在所有条件为 True 的情况下未返回副本 (GH 39595)

  • 修复了 DataFrame.replace() 中的回归问题，即当 regex 为多键字典时会引发 IndexError (GH 39338)

  • 修复了 object 列中浮点数的表示未遵循 float_format 的回归问题，即当在控制台中打印或通过 DataFrame.to_string()、DataFrame.to_html() 和 DataFrame.to_latex() 输出时 (GH 40024)

  • 修复了 NumPy ufunc（如 np.add）未向 DataFrame 传递所有参数的回归问题 (GH 40662)
