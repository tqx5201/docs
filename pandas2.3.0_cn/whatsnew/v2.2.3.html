2.2.3 版本新特性 (2024 年 9 月 20 日)#

以下是 pandas 2.2.3 中的更改。有关包括其他 pandas 版本的完整更新日志，请参阅发布说明。

Pandas 2.2.3 现已兼容 Python 3.13#

Pandas 2.2.3 是第一个普遍兼容即将发布的 Python 3.13 的 pandas 版本，并且此版本将上传适用于自由线程和普通 Python 3.13 的 wheel 文件。

和往常一样，请将发现的任何错误报告到我们的问题跟踪器

错误修复#

  • 在eval() 函数中，对包含除法 / 的complex 类型操作时会丢弃虚部。 (GH 21374)

  • 针对 numpy 2.1 兼容性的小修复。 (GH 59444)

其他#

  • 已将缺失的第三方依赖许可证重新添加到 wheel 文件中。 (GH 58632)
