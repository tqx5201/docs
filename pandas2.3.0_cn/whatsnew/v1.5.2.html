1.5.2 版本新特性 (2022 年 11 月 21 日)#

这是 pandas 1.5.2 版本中的更改。请参阅 发布说明 以获取完整的变更日志，包括 pandas 的其他版本。

修复的回归#

  • 修复了 MultiIndex.join() 在扩展数组数据类型中的回归 (GH 49277)

  • 修复了 Series.replace() 在数值数据类型和指定 value=None 时引发 RecursionError 的回归 (GH 45725)

  • 修复了 DataFrame 的算术运算在 MultiIndex 列具有不同数据类型时的回归 (GH 49769)

  • 修复了 DataFrame.plot() 阻止 Colormap 实例通过 colormap 参数传递的问题，如果使用了 Matplotlib 3.6+ (GH 49374)

  • 修复了 date_range() 在 CustomBusinessDay 频率和带有时间戳的 start 日期返回无效周期集的回归 (GH 49441)

  • 修复了 groupby 操作中的性能回归 (GH 49676)

  • 修复了 Timedelta 构造函数在子类化 Timedelta 时的回归，该回归导致返回错误类型的对象 (GH 49579)

错误修复#

  • 修复了写入时复制 (Copy-on-Write) 实现中在某些链式索引情况下视图跟踪丢失的问题 (GH 48996)

  • 修复了 Styler.to_excel() 中的内存泄漏 (GH 49751)

其他#

  • 还原了在函数 DataFrame.plot.scatter() 中将 color 作为 c 的别名以及 size 作为 s 的别名 (GH 49732)
