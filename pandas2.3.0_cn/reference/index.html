API 参考#

本页概述了所有公开的 pandas 对象、函数和方法。在 pandas.* 命名空间中公开的所有类和函数都是公开的。

以下子包是公开的。

  • pandas.errors: 由 pandas 引发的自定义异常和警告类。

  • pandas.plotting: 绘图公开 API。

  • pandas.testing: 用于编写涉及 pandas 对象的测试的函数。

  • pandas.api.extensions: 用于扩展 pandas 对象的函数和类。

  • pandas.api.indexers: 用于滚动窗口索引器的函数和类。

  • pandas.api.interchange: DataFrame 交换协议。

  • pandas.api.types: 数据类型类和函数。

  • pandas.api.typing: 可能需要用于类型提示的类。这些类是作为中间结果遇到的，但用户不应该直接实例化它们。这些类不要与来自 pandas-stubs 包的类混淆，该包除了 pandas 中出现的类之外，还包含用于类型提示的类。

此外，文档中提到了 pandas.io 和 pandas.tseries 子模块中的公共函数。

警告

顶级模块 pandas.core、pandas.compat 和 pandas.util 是私有的。这些模块中稳定的功能不受保证。
