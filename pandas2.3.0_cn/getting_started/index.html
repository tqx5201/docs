快速入门#

安装#

使用 conda？

pandas 是 Anaconda 发行版的一部分，可以使用 Anaconda 或 Miniconda 安装

conda install -c conda-forge pandas
偏好 pip？

pandas 可以通过 pip 从 PyPI 安装。

pip install pandas
需要深入说明？

安装特定版本？从源码安装？请查看高级安装页面。
